# Script: setup_env.sh

Путь: `scripts/setup_env.sh`.

## Назначение

- создать `.venv` (или использовать существующую),
- установить python зависимости,
- создать launcher `archviz` в `.venv/bin`.

## Использование

```bash
bash scripts/setup_env.sh
```
