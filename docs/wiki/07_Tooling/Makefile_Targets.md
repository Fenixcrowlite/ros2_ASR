# Makefile Targets

Основные цели:

- `make setup`
- `make build`
- `make test-unit`
- `make test-ros`
- `make test-colcon`
- `make test`
- `make run`
- `make bench`
- `make report`
- `make arch-static`
- `make arch-runtime`
- `make arch`
- `make arch-diff`
- `make clean`
- `make dist`

См. исходник: `Makefile`.
