# Repository Layout

- `ros2_ws/src` — ROS2 пакеты.
- `configs` — runtime profiles.
- `data` — sample wav + transcripts.
- `scripts` — automation scripts.
- `tests` — unit/integration.
- `tools/archviz` — auto architecture pipeline.
- `docs` — документация и диаграммы.
