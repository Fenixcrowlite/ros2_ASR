# How To Read This Wiki

## Принцип

Вики построена как сеть MOC-страниц (Map of Content):

1. Открываешь индекс раздела.
2. Переходишь в узкий кирпичик.
3. Возвращаешься по ссылкам вверх/вбок.

## Правило терминов

Если термин неочевиден, открой [[00_Start/Glossary]].

Для точечной навигации используются ссылки вида:

- `[[00_Start/Glossary#Backend]]`
- `[[00_Start/Glossary#Language Mode]]`
- `[[00_Start/Glossary#Topic asr_text_plain]]`

## Что считать "кирпичиком"

Кирпичик = отдельный технический элемент:

- нода,
- сервис,
- backend,
- конфиг,
- скрипт,
- этап пайплайна,
- тестовый контур.

Каждый такой элемент описан отдельной страницей в своей папке.
