# Quick Start Commands

## Полный базовый цикл

```bash
cd /home/fenix/Desktop/ros2ws
make setup
make build
make test-unit
make run
```

## Live speech (2 окна)

```bash
./scripts/open_live_test_terminals.sh
```

## Метрики и бенч

```bash
make bench
make report
```

## Архитектурные диаграммы

```bash
make arch
```

## Web GUI

```bash
make web-gui
```
