# Thesis Scope Rules

- The repository supports a bachelor thesis prototype for ROS2 ASR integration and benchmarking.
- Supported thesis context: Ubuntu 24.04, ROS2 Jazzy, COCOHRIP-like robotic/lab environment.
- Final benchmark evidence must distinguish implemented providers, smoke-tested providers, benchmarked providers, and skipped providers.
- Local and cloud providers may both be implemented, but cloud providers must appear in benchmark result tables only when actually benchmarked.
- Demo or smoke datasets must not be described as final benchmark datasets.
