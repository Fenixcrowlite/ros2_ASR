# Codex Director Rules

- Execute only the current task described in `.ai/tasks/current_task.md`.
- Do not perform unrelated refactors.
- Do not fabricate benchmark results.
- Do not use mock, fake, dummy, or synthetic providers in final thesis benchmark tables.
- Do not commit secrets or print secret values.
- Every final table must trace back to real run artifacts.
- Dataset validation must pass before final benchmark export.
- If a provider, dataset, or benchmark step is blocked, record the reason explicitly in `.ai/reports/current_task_report.md`.
