# Obsidian Setup

## Открытие vault

1. Open folder as vault.
2. Выбрать `docs/wiki`.
3. Открыть [[00_Start/Wiki_Home]].

## Рекомендуемые режимы

- Graph View: видеть связи страниц.
- Backlinks: видеть входящие ссылки на кирпичик.
- Outline: быстро прыгать по секциям страницы.

## Mermaid

Диаграммы проекта лежат в `docs/arch/*.mmd`.
Открой:

- `docs/arch/mindmap.mmd`
- `docs/arch/flow.mmd`
- `docs/arch/seq_recognize_once.mmd`

## Обновление диаграмм

```bash
make arch
```
