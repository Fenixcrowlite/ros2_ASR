# Script: make_dist.sh

Путь: `scripts/make_dist.sh`.

## Назначение

- создаёт release archive в `dist/`,
- исключает build/cache/venv/git/секреты,
- включает sources/docs/scripts/results/sample data.

## Использование

```bash
bash scripts/make_dist.sh
```
