# Script: open_live_test_terminals.sh

Путь: `scripts/open_live_test_terminals.sh`.

## Назначение

Открывает 2 терминала:

1. запуск ASR,
2. echo `/asr/text/plain`.

## Параметры

- `config`
- `input_mode`
- `continuous`
- `mic_capture_sec`

## Использование

```bash
./scripts/open_live_test_terminals.sh
```
