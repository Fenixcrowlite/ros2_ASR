# asr_interfaces

ROS2 interface package for runtime ASR, provider orchestration, dataset import, benchmark execution, and reporting workflows.

## Scope
- Messages for audio flow, ASR normalized output, status/events, benchmark summaries, and artifact references.
- Services for runtime control, profile/config access, dataset registry, and benchmark status.
- Actions for streaming sessions, benchmark runs, dataset import, report generation, and model download.
